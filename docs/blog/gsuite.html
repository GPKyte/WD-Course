About

The Problem

Automating GSuite On-Boarding

Java application for running bulk on-boarding tasks. Reduced weekly time spent from ~1 hour to 5 minutes.

The Solution

In start-ups with high turn-around, like EBTH, the process of on-boarding new employees can be time consuming. In the case of Everything But The House, it was distracting from meaningful work. This was in part due to a lack of flexibility for bulk actions.

Since much of on-boarding was done manually, we had the opportunity to reduce the time-cost significantly through automation. After researching Google's GSuite API and the build tool Gradle, I developed a standalone Java application for this purpose. The tool parses csv files generated from the HR department's on-boarding process and automatically performs the work of account creation, email group addition, and reaches out to the new employees with instructions for using their new account.

 

I later tied this project to a HTML signature generator which reads url parameters to auto-fill a simple form and create company branded email signatures.
